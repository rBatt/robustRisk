#robustRisk
---
An R package and analysis to understand whether the robustness of ecological communities to 3 classes of risk is influenced by 3 features of network structure.  

### Network Features:  
 1. Modularity
 2. Redundancy
 3. Diversity
 
### Classes of Risk:  
 1. Contagion
 2. Fat tails
 3. Collapse
